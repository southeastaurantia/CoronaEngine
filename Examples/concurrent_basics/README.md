# 并发工具集基础示例

这个示例程序展示了 Corona 并发工具集的基本用法，包括：

## 功能展示

### 1. 原子操作示例
- 基本的原子加载和存储
- 原子递增操作
- 比较并交换（CAS）操作

### 2. 自旋锁示例
- 多线程环境下的共享计数器保护
- RAII 锁守卫的使用
- 线程同步和竞争条件避免

### 3. 读写锁示例
- 多读者单写者的并发访问模式
- 读者和写者的协调机制
- 性能优化的共享数据访问

### 4. 内存分配器示例
- 高性能内存分配
- 缓存线对齐内存分配
- 内存使用统计

### 5. 线程工具示例
- CPU 信息获取
- 线程 ID 管理
- 高精度性能计时

## 构建和运行

这个示例会自动包含在 CoronaEngine 的构建系统中。

```bash
# 构建项目（包含示例）
cmake --preset ninja-mc
cmake --build --preset ninja-debug --target Corona_concurrent_basics

# 运行示例
./build/Examples/Debug/Corona_concurrent_basics.exe
```

## 预期输出

程序会输出各个组件的测试结果，包括：
- 原子操作的执行结果
- 多线程计数器的正确性验证
- 读写锁的并发访问日志
- 内存分配统计信息
- 系统性能指标

## 性能特点

- **高吞吐量**：设计目标为百万级 ops/s
- **低延迟**：微秒级响应时间
- **内存高效**：缓存友好的数据结构
- **可扩展性**：支持多核并行处理
